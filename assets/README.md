# Asset Policy

Images can help this project, but they should be used sparingly and safely.

## Good Assets

- cropped screenshots of the public-safe software interface;
- clean diagrams redrawn for this repository;
- synthetic CCD/measurement plots;
- equipment photos with no people, no private labels, and no sensitive lab
  records visible.

## Avoid

- raw experiment reports;
- full-resolution private screenshots;
- photos containing faces, names, student IDs, phone numbers, or lab records;
- unpublished hardware drawings or instrument design details unless publication
  permission is clear;
- videos and large raw image dumps.

Recommended first release: code plus one or two cropped screenshots only if
they make the workflow easier to understand.
